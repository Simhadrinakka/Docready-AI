# 📄 Docready-AI

> AI-powered document readiness verification platform for government schemes and scholarships.

## 🌟 Overview

Docready-AI helps applicants verify whether their documents are complete and ready before applying for government schemes. Instead of discovering missing or invalid documents after submission, users receive an AI-powered readiness report with recommendations, reducing application errors and improving success rates.

---

## 🚀 Features

- 🔍 AI-powered OCR document extraction
- 📑 Automatic document classification
- ✅ Document validation
- 📊 Readiness score generation
- 💡 Personalized recommendations
- 📋 Dynamic scheme-wise document requirements
- 📄 Professional report generation
- 🏛️ Government scheme support
- 📱 Responsive modern dashboard

## ⚙️ How It Works

1. Select a Government Scheme
2. Upload Required Documents
3. AI extracts document text using OCR
4. Documents are classified automatically
5. Validation checks are performed
6. Readiness score is calculated
7. Missing documents are identified
8. Recommendations are generated
9. Download the final verification report

## 💻 Local Installation

### Clone Repository

```bash
git clone <https://github.com/Simhadrinakka/Docready-AI>
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
